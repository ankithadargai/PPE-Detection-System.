Overview

The PPE Detection System is a computer vision-based project designed to improve safety in construction and industrial environments. It automatically detects whether workers are wearing essential safety equipment such as helmets, masks, gloves, and vests.

This system uses the YOLO deep learning model trained on datasets from Roboflow to ensure accurate and real-time detection.

Features:

Real-time PPE detection     
Supports image and video input    
Powered by YOLO, object detection

⚠️Identifies safety violations instantly     
📊 Visual bounding boxes for detection      
💡 Easy to integrate with surveillance systems

How It Works:

Input Source
Accepts images or live video feed from cameras
Detection
YOLOv8 model processes the input
Detects persons and checks for PPE
Output
Highlights individuals without PPE
Displays results with bounding boxes


visualisation:
